###### 5. Write the `correct answer` from the following options and give an explanation (2-5 lines).

```javascript
let count = 0;
const nums = [0, 1, 2, 3];

nums.forEach((num) => {
  if (num) count += 1;
});

console.log(count);
```

- A: 1
- B: 2
- C: 3
- D: 4

<details><summary><b>Answer</b></summary>
<p>

#### Answer: C

<i>
- It initializes a variable count with a value of 0.
- It defines an array called nums containing the numbers [0, 1, 2, 3].
- It uses the forEach method to iterate through each element in the nums array.
- Inside the loop, it checks if the current element (num) is truthy (i.e., not equal to 0).
- If the current element is truthy, it increments the count variable by 1.
- After looping through all elements, it prints the final value of count to the console.
- Since there are three truthy elements (1, 2, and 3) in the nums array, the final value of count is 3, which is printed to the console
</i>

</p>
</details>
